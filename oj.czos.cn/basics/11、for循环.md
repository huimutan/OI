## 一、什么是for循环

1、for的基本结构以及和while循环的对比

> 从m循环到n之间所有数的while循环有三个组成部分：初始值、循环条件、让循环停止 的语句,如循环输出1~10的每个数；

```cpp
#include <iostream>
using namespace std;
int main () {
    int i;//循环用的变量
    //初始值
    i = 1；
    //循环条件(在i<=10的条件下循环)
    while(i <= 10) {
        cout << i << endl;
        // i自增，让循环有停下来的可能
        i++ ；
    }
    return 0;
}
```

为了方便表达，我们将这三个组成部分放在一起来表达，就是for循环：

```cpp
for (初始值；循环条件；让循环停止的办法)
{
	语句；
}
```

课堂案例：输出1~ 10的每个数，用for循环应当这样写

```cpp
int i; //循环变量
//循环1 ~10
for (i = 1; i <= 10; i++)
{
    cout << i << endl;
}
//循环10 ~l
for (i = 10; i >= 1; i--)
{
    cout << i << endl;
}
```

2、理解for循环的执行顺序！ 

```cpp
for (语句1;语句2;语句3) {
    语句4;
}
```

先执行语句1, 且执行1次；

然后循环执行：语句2->语句4-〉语句3……，直到语句2的判断不成立!

注意理解for循环的执行顺序：

(1)    按照上述知识点中关于语句执行过程的讲解，可以看出首先执行i=1,为i赋初值；

(2)   接下来执行 i<=10 的判断条件；

(3)   如果条件成立，接下来执行cout << i << endl ;

(4)   执行i++,然后继续判断 i<=10是否成立；

(5)   请特别注意，循环结束后 i 的值是多少？

```cpp
int i; //循环变量
for (i = 1; i <= 10; i++)
{
    cout << i << endl;
}
cout << "循环结束 i=" << i << endl; // 11
```

(6)理论上for循环中分号隔开的3条语句都可以没有，但2个分号要保留，代码如下 (注意在实际编程中，虽然以下代码语法正确，但并不容易理解因此尽量不要这样写)。

```cpp
int i = 1; //循环变量
//循环1~10
for (; i <= 10;)
{
    cout << i << endl;
    i++;
}
```

#### 1264:【入门】4位反序数



```cpp
/*1264 - 【入门】4位反序数
题目描述
设N是一个四位数，它的9倍恰好是其反序数，求N。反序数就是将整数的数字倒过来形成的整数。例如：1234的反序数是4321。
输入
无
输出
输出N这个四位数
来源
简单循环
标签
简单循环*/
/*
    思路：
    第一步：循环所有的4位数(1000~9999)
    第二步：求出每个4位数的反序列
    拆出每个数的个位、十位、百位、千位，然后求倒数
    第三步：判断如果该数的9倍是其反序列再输出
    */
int i, g, s, b, q, x;// x：存放i的反序列
for (i = 1000; i <= 9999; i++){
    //拆位：i = 1234
    q = i / 1000% 10;
    b = i / 100 % 10;
    s = i / 10  % 10;
    g = i / 1   % 10;
    //测试拆位
    // cout<<q<<" "<<b<<" "<<s<<" "<<g<<endl;
    x = g*1000 + s*100 + b* 10 + q;
    //如果该数的9倍是其反序列x % i ==0 && x / i == 9
    if (i*9 == x){
        cout << i << endl;
    }
}
```

#### 1085:【入门】寻找雷劈数

```cpp
/*1085 - 【入门】寻找雷劈数
题目描述
把整数3025从中剪开分为30和25两个数，此时再将这两数之和平方，计算结果又等于原数。
(30+25)*(30+25)=55*55=3025，这样的数叫“雷劈数”。
求所有符合这样条件的四位数。 (ab+cd)*(ab+cd)=abcd

输入
无

输出
若干行，每行一个雷劈数，从小到大输出。

来源
需要找规律的循环

标签
需要找规律的循环
*/

/思路：
    第一步：i =循环1000-9999之间的每个数
    第二步：拆出每个数i的前两位和后两位
    i = 3025
    前两位a = i / 100
    后两位b = i % 100
    第三步：判断是否满足条件(a + b)  (a + b) == i 
    /
int i, a, b;
//循环 1000-9999
for (i = 1000; i <= 9999; i++)
{
    //拆出前两位和后两位
    a = i / 100;
    b = i % 100;
    if ((a + b)*(a + b) == i)

        cout << i << endl;

```

## 三、for循环和while循环的应用场景

1、 for循环 ：如果能够明确循环范围是从m~n,则更适合用for循环；

2、 while循环 ：如果没有明确的循环范围，但有循环条件，更适合用while循环；

1244:【入门】请问一个正整数能够整除几次2?

思路：

当n能够整除2时，计数器自增1,且n = n / 2;

while (n能够整除2) {

n = n / 2;

计数器自增1；

}

```cpp
int n, c; // n代表整数，c计数器
cin >> n;
c = 0;
//当n能够整除2 (偶数)
while (n % 2 == 0)
{
    n = n / 2;
    c = c + 1; //计数器自增
}
cout << c << endl;
```

1261:【入门】韩信点兵

```CPP
/*
1261 - 【入门】韩信点兵
题目描述
韩信有一对士兵，他想知道有多少人，他就让士兵报数，如果按照1到5报数，最末一个士兵报的数为1；按照1到6报数，最末一个士兵报的数为5；按照1到7报数，最末一个士兵报的数为4；最后再按1到11报数，最末一个士兵报的数为10，请问韩信这队士兵最少有多少人？

输入
无

输出
输出这队士兵最少有多少人
*/
```

思路：

从1开始找这样的一个数，满足：i % 5==1 && i % 6 == 5 && i % 7 == 4 && i % 11 ==10, 如果 i 不满足条件则继续找下一个！

上述整理的条件是一个循环停止条件：满足该条件循环停止！

反过来说，循环条件应该是：i%5 !=1 || i%6 !=5 || i%7 != 4 || i%11 !=10  也可以 !(i % 5 == 1 && i % 6 == 5 && i % 7 == 4 && i % 11 == 10)

```cpp
int i = 1;
//当士兵人数不满足条件，则循环
while (!(i % 5 == 1 && i % 6 == 5 && i % 7 == 4 && i % 11 == 10))
    // while (i % 5 != 1 || i % 6 != 5 || i % 7 != 4 || i % 11 != 10)
{
    i++;
}
cout << i << endl;
```

#### 1119:【入门】求各位数字之和(讲解短除法)

思路：采用短除法，n % 10得到当前的个位，然后n = n / 10去掉这个个位，重复这个 过程，直到n为0,循环停止！

```cpp
int n, s = 0;
cin >> n;
//当n!=0则循环
while (n != 0)
{
    // cout<<n % 10<<endl;
    s = s + n % 10;
    n = n / 10;
}
cout << s << endl;
```
