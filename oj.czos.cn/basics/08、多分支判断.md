## 一、多分支判断的语法

多分支的基本语法结构
if （判断条件）{
执行语句；
} else if （判断条件）{
执行语句；
} else if （判断条件）{
执行语句；
}…{
执行语句；
} else {
执行语句；
}
注意：
A、else if后面加判断条件，else后面不能加判断条件；
B、else if最后的else不是必须的；
C、多分支中至多只能有一个条件成立:也就是说一旦一个条件成立其余的条件不再判断;
